⚙️ Google pendiente de configurar
Sigue la guía para activar login y sincronización.
Paso 1 — Google Cloud
1. Ir a console.cloud.google.com
2. Nuevo proyecto → "Ferpa Panel"
3. Activar Google Sheets API y Drive API

Paso 2 — OAuth (Client ID)
1. Credenciales → Crear → OAuth → Aplicación web
2. Orígenes autorizados: poner URL de Netlify
3. Copiar Client ID → código línea CLIENT_ID

Paso 3 — API Key
1. Credenciales → Crear → Clave API
2. Copiar → código línea API_KEY

Paso 4 — Google Sheet
1. sheets.google.com → nueva hoja "Ferpa DB"
2. Crear pestañas: Config, Pedidos, Movimientos, Costos, Semanas, Jornadas, Plotter, Socios, Fondo
3. Copiar ID de la URL → código línea SHEET_ID

Paso 5 — Correos autorizados
Buscar CORREOS_AUTORIZADOS y poner los 3 Gmail reales

Paso 6 — Subir a Netlify
1. netlify.com → Sign up gratis
2. Arrastrar el HTML → obtener URL
3. Esa URL va en las credenciales OAuth (Paso 2)
✅ Con esto los 3 socios entran con su Gmail y comparten datos en tiempo real.
